Welcome!

Welcome to the Nordis sample web app homepage! This site only has a couple of pages, but it shows off a couple of features. First, however, notice that this web app utilizes a couple of client-side tools that are simply my personal preference. You can use whatever you want:

  1. Angular.js for client-side data-binding and API interactions (AJAX calls). There is additionally a home-brewed set of directives in the nordis.angular.0.0.5.js that help with those calls, as well as updating and paging through collections. Use and modify as you like, or roll your own.
  2. Require.js for asynchronous loading of javascript. Makes page-load feel faster, and on more complicated web apps you can load dependencies as needed for more intricate pages or apps.
This web app also demonstrates a couple of server-side features.

  1. Again, the template engine you use is up to you, but I use doT. I customized doT a bit to use double-braces [[ ]] instead of double-curly-braces for data-binding.
  2. I also added one helper function for including page-parts into a sever-side template. Check out the index.html code for how I included a separate header and footer.
  3. Page Stats. The web app is tracking hits and those hits are displayed on the stats page. Nordis supports configuring those stats you want to track, and then provides convenience methods for tracking stats, processing them and displaying them.
Get Started!

This web app does two things: 1) allows you to create a user record, and 2) view that record on the user details page. There is no security or session management here. It's bare-bones - on purpose.
