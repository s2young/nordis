Welcome!

Welcome to the Nordis sample web app! This site only has a couple of pages, but it demos a couple of key features.

  2. API Boilerplate. Again, the template engine you use is up to you, but I use doT. I customized doT a bit to use double-braces [[ ]] instead of double-curly-braces for data-binding.
  3. I also added one helper function for including page-parts into a sever-side template. Check out the index.html code for how I included a separate header and footer.
  4. Page Stats. The web app is tracking hits and those hits are displayed on the stats page. Nordis supports configuring those stats you want to track, and then provides convenience methods for tracking stats, processing them and displaying them.

The tools you use on the client-side of your web app are your choice. Nordis is on the server-side. However, this sample web app uses a few tools that I personally prefer that you may be interested in:

  1. Angular.js for client-side data-binding and API interactions.
  2. There is additionally a home-brewed set of directives and helper functions here that help with those calls, as well as updating and paging through collections. Use and modify as you like, or roll your own.
  3. Require.js for asynchronous loading of javascript. Makes page-load feel faster, and on more complicated web apps you can load dependencies as needed for more intricate pages or apps.
